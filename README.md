# Devops First Project

This is my first DevOps project built in Week 1.

## What I learned
- Git and GitHub
- Creating a repository
- CI/CD pipeline with GitHub Actions
- Deploying a live website with GitHub Pages

## Getting started
1. Clone the repository: git clone <repo-url>
2. Explore workflows in .github/workflows for CI/CD examples.
3. Visit the GitHub Pages URL in the repository settings to see the deployed site.
